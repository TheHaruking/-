# 遊戯王 オリジナルルール2

## 概要

遊戯王に、  
- トランプのお手軽さ
- 属性による相性
- HPやコスト(マナ)

などを取り入れたルールです。  
   
## ルールについて
| カードの種類    | 枚数
|:----------------|:-:
| モンスタカード  | 32
| 魔法カード      | 8
| 罠カード        | 12
- 合計52枚のカード。([ カードリスト ](デッキ.md))
  - 2人で1つの山札を使って遊びます。  

- 山札から引いたカードで勝負していき、**先に相手プレイヤに6回攻撃した者が勝利！**

## ゲームの流れ
1. 自分の番の開始。山札からカードを一枚引く。
   同時に、総ターン数に応じたコストを得る。
   - 1番初めなら"1"のコストを得る。5ターン目なら"5"のコストを得られる。
   - 先行の人は、一番初めのドローはしない。
1. 引いた後、コストを払ってモンスタ・罠を置いていく。(メインフェイズ)
   - 魔法も使える。
   - ゲームの後半になるにつれて、コストが多く得られるので、  
     強いモンスタが出せるようになったり、モンスタが沢山同時に出せるようになる。 
1. 置いた後、自分のモンスタで、相手モンスタ、または相手プレイヤを攻撃していく。(バトルフェイズ)
   - 魔法も使える。
1. 攻撃を終えたら、相手のターンへ。
1. 相手も同じように、
   1. カードを引き、コストを得て、
   1. 得たコストを払ってカードを置き、
   1. 置かれていたカードで攻撃して、
   1. ターン終了。
     - 相手の番では基本的に何もできません。
       ただ、罠を仕掛けていれば、相手の攻撃の際に妨害が可能。
1. 相手が終わったら、再び、自分の番の開始。

## 説明
- 目的     :  
  - 敵に**6**ポイント分攻撃。  
  - 8000あるLPを削っていくのではなく、6回分、相手プレイヤへの攻撃を決めれば勝利。  
  - お互いのプレイヤは、サイコロを使って残りポイントを管理するとよい。
- 攻撃     :  
  - 召喚したモンスタで、**相手プレイヤ** または **相手モンスタ** に行う。  
  - 相手に守備モンスタが1体でもいる場合、攻撃対象に選べるのは**守備モンスタのみ**となる。
  - 攻撃したくない場合、攻撃しなくてもよい。
- 召喚     :
  - モンスタはコストを払い、表側で**攻撃**表示(縦)、または表側で**守備**表示(横)で場に出す。
  - 出したターンは攻撃できない。(召喚酔い)
  - **召喚後、表示形式は変更できない**。
    - つまり、守備表示で出したモンスタはそれ以降攻撃できない。  
      攻撃表示で出したモンスタも同じく、後から守備表示に変えることができない。
- コスト : 
  - カードを場に出す際、コストを支払う必要がある。
  - 支払うためのコストは、自分の番の初めに、**現在の総ターン数**分 得られる。  
    - ゲーム開始時は"1"、次の自分の番では"2"得られる。
  - ただし、**罠カードを設置**している場合、  
    得られるコストは設置している枚数分 **-1** される。
    - もし、1ターン目に罠カードを伏せていたら、2ターン目に得られるコストは"1"となり、"2"とならない。
    - 2枚の罠カードを伏せていれば、得られるコストは-2される。
    - 次の自分の番までに伏せた罠を使用することが出来れば、コストを減らされない。
- 魔法  :
  - 自分の番に3コスト払って使える強力なカード。
  - 墓地のモンスタを復活させたり、敵全員にダメージを与えたりできる。
- 罠   :
  - 自分の番に1コスト払って設置できるカード。
  - 多くは「相手が攻撃したとき」に反応して、敵の攻撃を跳ね返したりすることができる。
  - カードに「相手が攻撃したとき」「相手が発動したとき」などと、  
    発動タイミングが特に指定されていない一部の罠カードは、  
    設置後すぐに使ってもよい。

### カードごとのコスト表

| 種類     | コスト |
|:-------- |:------:|
| 星2以下  |  1     |
| 星3      |  2     |
| 星4      |  3     |
| 星5      |  4     |
| 星6以上  |  5     |
| 罠       |  1     |
| 魔       |  3     |
- 星2以下のカードと罠は、1ターン目からすぐに使うことができるが、  
  星6以上のカードは、ゲームの中盤以降から使えることとなる。

## モンスターについて
- 攻撃力      :  
  - 敵のHP(守備力)を減らす能力
- 守備力      :  
  - HPとして扱う。これが0になると破壊される。
- 属性        :  
  - 三属性+二属性の計五属性がある。  
    それぞれ有利、不利が設定されている。  
    - 三大属性 [ 炎 → 風 → 水 → ... ]
      - 炎 は 風 に 有利。 水 に 不利。
      - 風 は 水 に 有利。 炎 に 不利。
      - 水 は 炎 に 有利。 風 に 不利。
        - 同じ属性同士の場合でも、有利不利はない。
    - 二大属性 [ 光 ⇔ 闇 ]
      - 光 は 闇 を攻撃するとき 有利。また、闇 に攻撃されるとき 不利。
      - 闇 は 光 を攻撃するとき 有利。また、光 に攻撃されるとき 不利。
        - **光同士、闇同士の戦闘の場合はお互い不利同士となり、どちらも攻撃力が減る。**


## 戦闘について
### モンスタを攻撃するとき
  - 有利属性のモンスタの攻撃力は500アップ。 
    - (例1-1) *"サンダー・キッズ(風)" が "海竜神(水)" へ 攻撃！* 
      - 攻撃力700 + 500 で、1200ダメージ！
    - (例2-1) *"運命のろうそく(闇)" が "青眼の白龍(光)" へ 攻撃！*
      - 攻撃力600 + 500 で、1100ダメージ！

  - 不利属性のモンスタの攻撃力は**500になる**。  
    また、900以下だった場合、500ダウン。
    - (例1-2) *"海竜神(水)" の "サンダー・キッズ(風)" への 反撃！*
      - 攻撃力1800ですが、500ダメージの反撃しか与えられません。  
        - 後述しますが、戦闘はお互いが攻撃し合います。
    - (例2-1) *"青眼の白龍(光)" の "運命のろうそく(闇)" への 反撃！*
      - 攻撃力3000 の"青眼~"ですが、"運命~"に、500ダメージしか与えられません。
      - 光⇔闇の戦闘のとき、反撃側が不利となります。
    - (例3-1) *"ホーリー・エルフ(光)" が "北風と太陽(光)" へ 攻撃！*
      - "ホーリー~"の攻撃力は 800 - 500 で 300 になります。
    - (例3-2) *"北風と太陽(光)" の "ホーリー・エルフ(光)" への 反撃！*  
      - 反撃側、"北風~"も、 攻撃力1000 から 500 にダウンします。
    
### 戦闘手順
  1. 攻撃対象を選択する。ただし、守備モンスタがいれば、守備モンスタしか選択できない。
  1. 属性を確認する。有利不利があれば、それに応じて自分モンスタ、相手モンスタの攻撃力を計算する。
  1. 発動したい罠があれば、ここで発動してもよい。
  1. 自分モンスタの攻撃。
     相手モンスタへ、自分の攻撃力分のダメージを与える。  
     同時に、相手モンスタも反撃を行う。
     自分のモンスタに、相手モンスタの攻撃力分のダメージを与える。
     -  サイコロや硬貨を乗せてカウントするとよい。
  1. 受けたダメージが守備力を上回ったら、そのモンスタは墓地に送られる。
    
### 相手プレイヤを攻撃する場合
- 攻撃力に関係なく、モンスタは相手プレイヤに1ポイント分のダメージを与える。  
  - ただし、星6以上のモンスタは2ポイント分のダメージを与える。
- 攻撃された相手は、カードを一枚引く。
  - **もしそれが魔法カードなら、コストを支払わず使用してもよい。**
  - 星6以上のモンスタに攻撃された場合でも、引けるのは1枚。

## エラッタ
- 昼夜の大火事
  - 元 : 相手のライフポイントに800ポイントのダメージを与える。
  - 改 : 相手のフィールド上モンスター全てに800ポイントのダメージを与える。
        
- 魔法の筒(マジック・シリンダー)
  - 元 : 相手モンスター1体の攻撃を無効にし、  
         相手のライフポイントにそのモンスターの攻撃力分のダメージを与える。
  - 改 : 相手の攻撃時に発動可能。  
         相手モンスターを1体選択し、攻撃をそのモンスターへと反射させる。  
         (選択する際、表示形式は問わない。属性計算も適用される。  
         反射による攻撃を受けたモンスターの反撃は行われない。)
             
- あまのじゃくの呪い
  - 元 : ターン終了まで、パワーアップ・ダウンの効果は逆に作用する。
  - 改 : ターン終了まで、属性の優劣は逆に作用する。  
         (本来不利属性となる側がパワーアップし、有利属性はパワーダウンする。)

- クリボー
  - 元 : 手札からこのカードを捨てる。  
    相手からプレイヤーへのダメージを1度だけ0にできる。
    この効果は相手のバトルフェイズしか使えない。
  - 改 : 手札からこのカードを捨てる。次から効果を1つ選び発動する。  
    ・ 相手モンスターからの自分モンスターへの戦闘ダメージを1度だけ0にする。  
    ・ 相手モンスターからのプレイヤーへの直接攻撃を1度だけ無効にする。  
    この効果は相手のバトルフェイズしか使えない。

## ルール要注意カード
- 和睦の使者
  - 効果 : 相手モンスターからの戦闘ダメージを、全て発動ターンだけ0にする。
    - 自分のターンに使うことで、攻めのときにも積極的に使えるカード。
- シフトチェンジ
  - 効果 : 相手が魔法・罠・戦闘で自分のフィールド上モンスター1体を指定してきた時に発動可能。  
    他の自分のモンスターと対象を入れ替える。
    - "魔法の筒"にも発動できる。
## その他変更
- スペルスピード廃止
  - 相手の"万能地雷グレイモヤ"にチェーンして、"援軍"を発動。  
    そして、自分の破壊されるモンスターを変える、ということが可能。
  - 相手の"あまのじゃくの呪い"にチェーンして、自分の"あまのじゃくの呪い"を発動。  
    そして、相性を元に戻す、ということも可能。
- 罠カード、伏せてからすぐ使用可。
  - "援軍"、"和睦の使者"といった、相手の行動が発動条件でないカードが使用可。

## 特別ルール
- "クリボー" は "青眼の白龍" を、 "薄幸の美少女" は "ブラック・マジシャン" を、  
  それぞれ無条件で倒せる。
  戦闘時、攻撃時も守備時も必ず勝利し、ダメージも受けない。
- 守備力500のモンスタは、出したターンに攻撃が可能。
  - スティング
  - ガーゴイル
  - はにわ
- 魔法の筒を、魔法の筒で返す。が可能です。
